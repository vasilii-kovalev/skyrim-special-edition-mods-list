### [Weapons Armor Clothing and Clutter Fixes](https://www.nexusmods.com/skyrimspecialedition/mods/18994)

#### Actions

* Download, install and enable ["Weapons Armor Clothing and Clutter Fixes" file](https://www.nexusmods.com/skyrimspecialedition/mods/18994?tab=files&file_id=306683&nmm=1). Choose the following options in FOMOD:
  * "Textures" page:
    * "Texture Resolution" section:
      * Select "2K (default)" option
  * "Patches" page:
    * "Orcish Weapons" section:
      * Select "None" option
    * "Bashed Patch" section:
      * Uncheck everything
  "Options" page:
    * "Optional Files" section:
      * Check **only** the following options (and uncheck others):
        * Vanilla Daedric Weapons
        * Vanilla Dragon Priest Masks
        * Vanilla Orcish Weapons
* Put the `*.esp` files in the following order in the "Plugins" tab:
  * Weapons Armor Clothing & Clutter Fixes
  * WACCF_VanillaOrcishWeapons
  * WACCF_VanillaDaedricWeapons
  * WACCF_VanillaDragonPriestMasks

#### Requires

* [Unofficial Skyrim Special Edition Patch]()

#### Comments

* "Vanilla Gold Coin" option was not check since I'm fine with the renaming
* "Vanilla Projectiles" option was not checked since I find the vanilla variant less convenient, especially when firing at long distances or firing fast at the approaching enemies

### [Scrambled Bugs](https://www.nexusmods.com/skyrimspecialedition/mods/43532)

#### Actions

* Download, install and enable ["Scrambled Bugs (Special Edition)" file](https://www.nexusmods.com/skyrimspecialedition/mods/43532?tab=files&file_id=269169&nmm=1)
* Make the following changes in `SKSE/Plugins/ScrambledBugs.json` file:
  * Change `attachHitEffectArt` field's value to `true`
  * Change `blackSoulGems` field's value to `true`
  * Change `powerAttackStamina` field's value to `true`
  * Change `steepSlopes` field's value to `true`
  * Change `underfilledSoulGems` field's value to `true`

#### Requires

* [Address Library for SKSE Plugins]()
* [Bug Fixes SSE]() (for `steepSlopes`)
* [Unofficial Skyrim Special Edition Patch]()
